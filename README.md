# Atividade - DOM CONCEITOS BÁSICOS

Este projeto foi desenvolvido como parte da disciplina de Desenvolvimento Web, com o objetivo de aplicar, na prática, conceitos fundamentais de manipulação do DOM utilizando JavaScript puro.

## Objetivo

Construir uma página interativa capaz de responder a ações do usuário, explorando eventos, criação dinâmica de elementos e atualização de conteúdo em tempo real.

## Funcionalidades

Contador de cliques:

* Permite incrementar e decrementar um valor exibido na tela
* Impede que o valor seja negativo
* Exibe um alerta ao tentar decrementar abaixo de zero

Entrada de texto dinâmica:

* O usuário pode digitar no campo de texto
* Ao pressionar Enter, o conteúdo é inserido como um novo parágrafo na página

Contador de caracteres:

* Atualiza em tempo real conforme o usuário digita
* Considera apenas caracteres não vazios (ignora espaços)

Lista dinâmica:

* Permite escolher entre lista ordenada e não ordenada
* Adiciona automaticamente itens ao clicar no botão

Reset da aplicação:

* Restaura o estado inicial da página
* Limpa contador, textos inseridos e listas criadas

## Conceitos aplicados

* Manipulação do DOM (seleção e criação de elementos)
* Tratamento de eventos (click, input, keypress)
* Estruturas de controle (condicionais e laços)
* Atualização dinâmica de conteúdo
* Organização do código em arquivos separados (HTML, CSS e JavaScript)

1. Clonar o repositório:

```

```

```
cd atividade-dom
```

3. Abrir o arquivo index.html em um navegador
